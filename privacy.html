Privacy Policy

Last updated: 21 May 2026

Overview

This application is a personal automation project that uses the Oura API to retrieve step count and activity-related data for the purpose of generating a daily steps and distance report.

Data collected

The application may access Oura activity data, including daily steps, activity summaries, timestamps, and related metadata returned by the Oura API.

How data is used

Data is used only to calculate and report daily steps and estimated distance. The application does not sell, rent, or share personal data with advertisers or data brokers.

Data storage

The application may store daily step counts, calculated distance, API response metadata, and authentication tokens where required for operation.

Data sharing

Daily report information may be sent to the configured email recipient. No other third-party sharing is performed except where required to operate the application, such as email delivery infrastructure.

Security

Reasonable technical measures are used to protect stored data and credentials. Secrets and access tokens should not be committed to public repositories.

User control

Access to Oura data can be revoked through the Oura account or developer settings.

Contact

For questions about this policy, contact the owner of this application.
